# Solo-jazz.nvim

Port of [Solo Jazz Emacs theme](https://github.com/cstby/solo-jazz-emacs-theme) for Neovim.

## Installation

With [lazy.nvim](https://github.com/folke/lazy.nvim):

``` lua
{
    "xbonj/solojazz.nvim"
}
```

## Usage

### Vim Script

``` vim
colorscheme solojazz
```

### Lua

``` lua
vim.cmd('colorscheme solojazz')
```
